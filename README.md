# homebrew-cursor-voice

Personal Homebrew tap for [Cursor Voice](https://github.com/cursorvoice/cursor-voice).

## Install

```bash
brew tap cursorvoice/cursor-voice
brew install --cask cursor-voice
```

The app is ad-hoc signed (no paid Apple Developer ID), so the cask's
`postflight` strips the quarantine attribute automatically — no
right-click-Open dance required.

See the main repo for source, build instructions, and release notes.
